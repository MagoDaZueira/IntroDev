# Projeto da Fase WebMAC

Neste diretório teremos todo o projeto da primeira fase da disciplina

A aplicação que criarei é um **site de digitação**, onde o usuário tenta digitar palavras aleatórias o mais rápido possível

Escolhi isso pois, além de ser um tema pelo qual me interesso, parece ter um escopo adequado, com informações interessantes para guardar no banco de dados (para cada usuário, suas tentativas, recordes), e peso razoável no frontend

Fora isso, é uma ideia que teria um bom potencial de expansão caso eu queira melhorá-la após o tempo contemplado pela disciplina
